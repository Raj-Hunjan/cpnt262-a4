# Simple Node Command Line App
A simple app where you have to choose which door to go through.
You can choose doors 1-5.

## Usage:
Enter a command in the following format:

```
$ node app.js <door> <number>
```

## Examples:
- `$ node app.js door 1`
- `$ node app.js door 2`

## Errors:
- `$ node app.js door`
- `$ node app.js 4`
